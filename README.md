# Retail-Market-Demand-Analysis


## Project Overview

This project focuses on analyzing retail sales data to uncover customer purchasing behavior, sales trends, regional performance, and high-performing product categories. The objective of this project is to help businesses make data-driven decisions by identifying important sales patterns and customer insights.

The project combines Python for data cleaning and analysis with Power BI for interactive dashboard visualization.

---

## Business Problem

Retail businesses often struggle to:

* Identify top-performing regions and products
* Understand seasonal sales trends
* Track customer purchasing behavior
* Make informed business decisions using raw sales data

This project solves these problems by transforming raw retail data into meaningful business insights through data analysis and visualization.

---

## Objectives

* Analyze monthly sales performance
* Compare sales across different regions
* Identify top-selling products and categories
* Generate actionable business insights
* Build an interactive dashboard for decision-making

## Project Workflow

1. Data Collection
2. Data Cleaning using Python
3. Date Formatting & Feature Engineering
4. Exploratory Data Analysis (EDA)
5. Sales Trend Analysis
6. Dashboard Creation in Power BI
7. Business Insight Generation

---

## Dashboard Features

* KPI Cards for Total Sales and Total Orders
* Monthly Sales Trend Analysis
* Region-wise Sales Comparison
* Product Category Performance
* Top 5 Selling Products
* Interactive Slicers and Filters

---

## Key Insights

* West region generated the highest overall sales performance.
* Technology products contributed the largest share of revenue.
* Sales showed noticeable seasonal trends across months.
* A small group of products generated a major portion of total sales.
